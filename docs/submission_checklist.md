# Submission Checklist

## Core deliverables

- Public GitHub repository with a working `README.md`
- 6-8 page report in the CVPR template
- arXiv upload of the final report
- `videos.zip` containing processed results for:
  - `wild`
  - `bmx-trees`
  - `tennis`

## Required report sections

- Abstract with GitHub repository link at the end
- Introduction
- Related Work citing all references from the project instruction PDF
- Method with a clear pipeline figure
- Experiments with tables and qualitative figures
- Conclusion with limitations and future work

## Experiment completeness

- Part 1 baseline runs on all mandatory datasets
- Part 2 runs on all mandatory datasets
- Part 3 includes at least one failure case and one improvement attempt
- DAVIS or another GT-backed dataset is included for strong quantitative evidence

## Metrics

- `JM / JR` for mask quality
- `PSNR / SSIM` for restoration quality when GT is available
- side-by-side figure comparisons for every method section in the report

## Final packaging

- Verify all links in the PDF work
- Verify the README run commands are current
- Verify the zip contains report, links, and processed videos
- Record the exact submission timestamp before the deadline
