\section{Introduction}
Video object removal aims to erase dynamic foreground objects while recovering a temporally consistent background.
This task is useful for scene cleanup, editing, robotics, and video understanding.
In this project, we compare a controllable classical baseline with a modern AI-driven pipeline, then analyze failure cases where the background is never visible.
